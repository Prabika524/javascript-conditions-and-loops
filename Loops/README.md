# JavaScript Loop Assignment

This project is a beginner level **JavaScript Loop Practice Assignment**.  
It includes different programs using **for loop** and **while loop** to understand basic JavaScript logic and console output.

---

## Project Structure
Assignment Web
Loops
 ├─ index.html
 ├─ loop.js
 └─ README.md

- For Loop
- While Loop
- Sum of Numbers
- Even & Odd Numbers
- Factorial
- Fibonacci Series
- Array Iteration
- Largest Number in Array
- Reverse Number
- Patterns (Stars & Numbers)

## How to Run (Browser Method)

1. Open the project folder in **VS Code**
2. Double-click **index.html**
3. Browser will open
4. Right Click → **Inspect**
5. Go to **Console Tab**
6. You will see all JavaScript outputs

---
